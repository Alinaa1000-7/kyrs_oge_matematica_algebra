Добро пожаловать на курс

ПО ПОДГОТОВКЕ К ОГЭ ПО МАТЕМАТИКЕ
(МОДУЛЬ АЛГЕБРА)

В этом курсе собрана вся необходимая информация, для успешной сдачи ОГЭ по математике.
Далее будет представлено содержание курса и при желании вы можете ознакомиться только с сложными в понимании для вас темами
Надеемся, что курс окажется полезен для вас, удачи в дальнейшей сдачи экзаменов. До новых встреч.

— Цель данного курса:

Основной целью курса является устранение основных проблем в знаниях учеников 7-9 классов и подготовка их к сдаче основного государственного экзамена (ОГЭ).
Для учеников 10-11 классов курс будет полезен в качестве повторения при подготовке к ЕГЭ.

— Формат курса:

Курс состоит из 8 модулей.
Каждый модуль состоит из нескольких уроков, которые содержат в себе минимальную необходимую теорию и тестовые задания.
Обычно каждый урок достаточно короткий, что позволяет пройти за 1 день приличное количество тем.
В конце каждого модуля вас ждёт тест по пройденым темам.
Несмотря на то, что вся нужная вам теория собрана в одном месте, рекомендуется вести свой собственный конспект, это поможет более лучшему усвоению информации.

  • Теория (Готовы первые два юнита)
  • Практика (Готова первая тема)
  • Справочные материалы ??
